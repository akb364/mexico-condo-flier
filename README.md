# Princesa Penthouse Flier

A simple one-page landing page for an owner-direct penthouse rental at Princesa
de Puerto Penasco.

## Edit checklist

- Replace `hello@example.com` and `(555) 012-3456` with the booking contact.
- Connect the buttons to Airbnb, VRBO, a booking form, or the client's email.
- Confirm the bathroom count and maximum guest count if those should be listed.

Open `index.html` in a browser to preview the page.
