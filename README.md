# Mexico Condo Flier

A simple one-page landing page for a client's Mexico rental condo.

## Edit checklist

- Replace `Casa Mar Azul` with the real property name.
- Update the headline, city, amenities, guest count, and beach distance.
- Replace `hello@example.com` and `(555) 012-3456` with the booking contact.
- Connect the buttons to Airbnb, VRBO, a booking form, or the client's email.

Open `index.html` in a browser to preview the page.
